Adds Twig Template syntax highlighting to SyntaxHighlighter Evolved

== Description ==

SyntaxHighlighter Evolved offers lots of popular language syntaxes and this 
plugin adds support for Twig Template code.

== Installation ==

This plugin depends on the [SyntaxHighlighter Evolved](http://www.viper007bond.com/wordpress-plugins/syntaxhighlighter/) plugin. 
See that plugin's website to get that plugin first.

After you have Syntax Highlighter Evolved installed, put this plugin in your plugins directory 
and activate it on your Plugins Admin page.
